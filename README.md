# Django Voting Platform

Профессиональная система онлайн-голосований, разработанная с фокусом на масштабируемость, безопасность и удобство управления данными. Проект построен по архитектуре **Backend-first** с использованием полноценного REST API. Проработана адаптивность для слабовидящих, для печати, и разработаны различные темы оформления.

---

##  Основной функционал

*   **Гибкое управление голосованиями:** Поддержка трех статусов: публичные, приватные (доступ по токену) и черновики (отложенная публикация).
*   **Безопасный REST API:** Система разграничения прав доступа — пользователи могут управлять только собственными объектами.
*   **Аналитика и экспорт:** Полная интеграция с Excel для выгрузки отчетов по итогам голосований.
*   **История изменений:** Благодаря `django-simple-history` сохраняется аудит всех действий: кто, когда и что изменил в системе.
*   **Медиа-поддержка:** Возможность загрузки изображений к номинациям и прикрепления PDF-руководств к каждому голосованию.

---

##  Технологический стек

### Back-end
*   **Framework:** Django 5.2.10, Django REST Framework
*   **Database:** PostgreSQL
*   **Data Handling:** `django-import-export`, `openpyxl`, `tablib`
*   **Utilities:** `django-filter`, `django-simple-history`, `pillow` (обработка изображений)

### Front-end
*   **Framework:** Vue 3, Vite
